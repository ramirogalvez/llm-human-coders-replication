# Output Data

Generated files are written here when the replication scripts are run.

- `plots/`: PDF figures generated by `../../src/full_analysis.py`.
- `tables/`: LaTeX and Excel tables generated by the Python and Stata scripts.
- `data_for_regs.txt`: tab-delimited file generated by `../../src/full_analysis.py` and used by `../../src/Replication_Package_Stata.do`.

Generated outputs are intentionally not included in the repository.
